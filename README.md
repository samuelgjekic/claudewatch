## Install

### Download (recommended)

Download the latest **ClaudeWatch.dmg** from [Releases](https://github.com/samuelgjekic/claudewatch/releases), open it, and drag ClaudeWatch to your Applications folder. Then run:

```bash
xattr -cr /Applications/ClaudeWatch
```

Open ClaudeWatch from Applications.

### Build from source

```bash
git clone https://github.com/samuelgjekic/claudewatch.git
cd claudewatch
swift build && swift run
```

Or build and install the .app bundle:

```bash
make install
```

## Features

- **Real-time usage tracking** — Session (5h window) and Weekly (7d window) usage pulled live from the Anthropic API
- **Dual battery icons** — See both session and weekly usage at a glance in your menu bar
- **Sonnet tracking** — Automatically shown for Max/Team subscribers
- **Display modes** — Switch between Session, Weekly, Both, or Sonnet in settings
- **Active sessions** — See your running Claude Code sessions with version, duration, and PID
- **Stats dashboard** — Lifetime stats, model breakdown with token counts, daily activity sparkline, and hourly heatmap
- **Dark UI** — Clean dark theme with color-coded usage bars (green/blue/orange/red)

## Requirements

- macOS 14+
- Any Claude subscription (Pro, Max, or Team)

## How It Works

1. Sign in via the built-in browser window (captures your claude.ai session)
2. ClaudeWatch polls the Anthropic usage API every 2 minutes
3. Usage is displayed as battery icons in the menu bar and detailed bars in the popover
4. Local stats are read from `~/.claude/stats-cache.json` (generated by Claude Code)

## Uninstall

```bash
make uninstall
```

Or just drag ClaudeWatch.app to the Trash.
